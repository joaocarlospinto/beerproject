This site is a personal project created while studying Angular and Java.

I started learning JAVA 17, then Spring Boot, and created some Rest API, but I could only see the results using POSTMAN

I thought, what about learning some front-end framework? So I could see the result better. Then I started learning ANGULAR.

The first part of the project was to create a REST API with CRUD operations on a table called 'Beers'.

The Front-End includes this about page and a list of Beers where you can access the details of each beer, update, and delete it.

I wanted to display the Beer's Image, so I created a new API to store and retrieve the beer's image by name.

Front End:

  • Angular - HTML, CSS, TypeScript and JavaScript
  • Angular Materials
The Front-End is deployed on Github Pages.
You can find the Front End code in this repository

Back End : 2 API Rest - The Beer database CRUD and the Image store/retrieve

  • Java
  • Spring Boot
  • JPA
  • PostGreSQL
  • Hateoas
The Back End is deployed on Render (Docker).

You can find the Back End Beer API code in this repository

You can find the Back End Image API code in this repository
